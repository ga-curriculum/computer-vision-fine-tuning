## About
Fine-tuning is a method of modifying generalized Computer Vision models (CVs) to have better performance in specific domains. 


## Learning Objectives

*You will be able to:*

- Describe fine-tuning use cases
- Explain the goals and tradeoffs of model fine-tuning 
- Apply fine-tuning of an image model with Hugging Face
